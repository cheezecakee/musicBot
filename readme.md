# musicBot

A simple discord bot that streams music. 

## Requirements
ffmepg
yt-dpl 

### Put these in a .env file:
* Discord Token
* Spotify ID
* Spotify Secret
* Youtube API key

### How to use
Use the / command to start using 
Available commands:
* Play
* Skip
* Prev
* Queue
* Pause (Currently Unavailable)
* Resume (Currently Unavailable)
